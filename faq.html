Frequently Asked Questions

How much does it cost to create a token?

The cost of creating a token is approximately 0.01 SOL.

How long does it take to create a token?

It takes less than a minute to create a token.

How do I verify my token on Solscan?

To verify your token on Solscan, you will need to submit a request to have it updated. You can do this by going to the token's page on Solscan and clicking the "Update" button. You will then need to fill out a form with the token's information, including the metadata URI.

How do I add liquidity on Raydium?

To add liquidity on Raydium, you will need to go to the "Liquidity" page and click the "Create Pool" button. You will then need to select the two tokens that you want to add to the pool and the amount of each token that you want to add. Once you have created the pool, you will be able to add liquidity to it by clicking the "Add Liquidity" button.
